Progressive Web Applications (PWAs)

A PWA (progressive web application) is a type of app that’s delivered on the user’s web browser. It employs a combination of common technologies, like HTML, CSS, JSON and JavaScript, to create a fast, feature-rich user experience that rivals that of a native app. PWAs work particularly well on the most modern browsers. According to Google, an application must use four things in order to be a PWA: a web app manifest, service workers, HTTPS and an icon (for the user's browser homepage).

PWAs employ a relatively new methodology to offer useful features and benefits due to being:

  • Responsive - They work on any device with a browser, whether mobile, desktop or emerging. Design for mobile first.
  • Sticky - Foster attraction and retention of user attention, using notifications and user experience on par with native apps.
  • Viral Ready - Easy to share via URL, and users don’t have to hassle with app stores or complex installation.
  • Easy to Access - User can opt to launch a PWA from a clickable icon on their browser home screen.
  • Fast and Reliable - Once a PWA has been loaded, caching allows high performance speed even when it’s offline or on a slow connection.
  • Secure and Up to Date - PWAs only work using HTTPS. Service workers handle updates, so the user doesn’t have to worry about it.
